## 🏢 협업 방식

| Tag Type     | content                       |
|--------------|-------------------------------|
| `✨feat`      | 기능(feature)                   |
| `📝docs`     | 문서 작업 (documentation)         |
| `🐛fix`      | 버그 수정                         |
| `🚑hotfix`   | 긴급 버그 수정                      |
| `♻️refactor` | 코드 리팩터링(가독성 향상)               |
| `👷ci`       | ci 파이프라인 구축 관련                |
| `💚ci/cd `   | ci 빌드 수정 및 deployment         |
| `chore`      | 관리(maintain), 핵심 내용은 아닌 기타 변경 |
| `test`       | 테스트                           |
| `style`      | 포맷팅, 세미콜론 누락 등                |
